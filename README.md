# HetmanInvest Compound Interest Calculator

Статический фронтенд-калькулятор сложного процента для HetmanInvest без сборки и внешних фреймворков.

## Файлы

- `index.html` — структура страницы, формы, результатов и CTA.
- `styles.css` — фирменная палитра, glass/card визуальный стиль и адаптивность.
- `script.js` — мгновенный пересчёт, сценарии и таблица динамики по годам.

## Запуск

Откройте `index.html` в браузере.
