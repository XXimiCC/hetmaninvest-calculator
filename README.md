# HetmanInvest Compound Interest Calculator

Статический фронтенд-калькулятор сложного процента для HetmanInvest без сборки, фреймворков и внешних chart-библиотек.

## Что добавлено

- Сравнительный SVG-график на чистом HTML/CSS/JavaScript.
- Линии для сценариев:
  - `Инвестировать сейчас`
  - `Просто откладывать без доходности`
  - `Старт через 2 года`
  - `Старт через 3 года`
- Компактный блок инсайтов с объяснением цены ожидания.
- Обновлённая русскоязычная summary-подача рядом с графиком.

## Файлы

- `index.html` — структура страницы, форма, результаты, SVG-график, легенда и блок инсайтов.
- `styles.css` — фирменная glass/card стилистика, оформление графика, легенды, карточек сравнения и адаптивность.
- `script.js` — пересчёт базового сценария и сравнительных вариантов, генерация данных и отрисовка SVG-графика.

## Логика сравнения

- `Инвестировать сейчас` — базовый сценарий с заданной доходностью.
- `Просто откладывать без доходности` — те же взносы и стартовая сумма, но `0%` годовых.
- `Старт через 2 года` и `Старт через 3 года` — тот же горизонт, но первые 2 или 3 года взносы и инвестиционный рост не работают.

## Запуск

Откройте `index.html` в браузере.
